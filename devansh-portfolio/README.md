# Devansh Portfolio

This is a standalone static portfolio website for Devansh Shah.

## Project Files

- `index.html` - page structure and content
- `assets/css/styles.css` - complete visual design and responsive layout
- `assets/js/main.js` - navigation and reveal interactions
- `assets/docs/Devansh-Shah-Resume.pdf` - downloadable resume

## Run Locally

Open `index.html` directly in a browser.

## Deploy

This site is ready for static hosting on platforms like Netlify or Vercel.

- Netlify: set the publish directory to `devansh-portfolio`
- Vercel: set the root directory to `devansh-portfolio`
